🔧

Extraction & Décompilation

Zip extractionJADX decompiler~5-30 secondes

📋

Analyse du Manifest

Permissions (23 dangereuses)Composants exportésSDK min/target

💻

Analyse du Code Java

Scan des fichiers .java décompilés avec 20+ règles : détection de code dangereux, secrets en dur, WebView vulnérable, SQL injection, crypto faible, logs sensibles.

🔬 Corrélation multi-signaux

L'IA analyse les combinaisons de permissions et comportements suspects

📱 SMS + 🌐 Internet + 🔑 Secrets

Exfiltration de données

Combinaison de permissions SMS, accès réseau et détection de secrets en dur → indicateur fort d'exfiltration.

♿ Accessibility + 📱 SMS

Spyware

Service d'accessibilité + lecture SMS → permet l'interception des messages et codes 2FA.

📷 Camera + 🎤 Micro + 🌐 Internet

Surveillance

Accès caméra/microphone + communication réseau → permet l'exfiltration de médias enregistrés.

🗄️ Storage + 🌐 Internet + 🔑 API Keys

Vol de données

Lecture stockage externe + réseau + clés API → exfiltration massive vers serveur distant.

📊 Comprendre les sévérités

Niveaux de vulnérabilités

Classification AHP pour prioriser les corrections

🔴 CRITICAL

  • ! Secrets en dur — extraction possible par attaquant
  • ! WebView RCE — exécution de code arbitraire
  • ! Debuggable en prod — exposition des données
  • ! SQL injection — accès DB non autorisé

🟠 HIGH

  • ! HTTP en clair — interception MITM
  • ! Permissions excessives — accès non justifié
  • ! allowBackup activé — fuite via sauvegardes
  • ! Composants exportés — accès tierces apps

🟡 MEDIUM

  • ! Crypto faible — MD5, SHA-1, DES, ECB
  • ! Logs sensibles — exposition en clair
  • ! java.util.Random — valeurs prévisibles
  • ! IV statique — affaiblissement CBC

🟢 LOW

  • ! Emails exposés — information mineure
  • ! Anti-debug détecté — code obfusqué
  • ! IPs internes — révélation infrastructure
